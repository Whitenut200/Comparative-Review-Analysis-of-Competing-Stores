## 프로젝트 개요
‘와우솥뚜껑삼겹살(주 가게)’과 인근 경쟁 가게의 리뷰 데이터를 분석하여 주 가게의 문제점과 강점, 차별화 포인트를 도출하는 것이 목표입니다.
네이버지도에서 수집한 리뷰 데이터를 기반으로 다양한 지표를 산출하고, 이를 Tableau 대시보드로 시각화하여 인사이트를 직관적으로 확인할 수 있도록 구현했습니다.
대시보드 개발 완료되면, 인사이트 도출 후 팀원과 마케팅 기획 및 실행할 예정입니다.

---

## 프로젝트 주제
- **주제:** 주 가게와 경쟁 가게의 리뷰를 비교해 **문제점·강점·차별화 포인트** 도출  
- **기간:** 총 3.5주  
  - 데이터 수집 및 전처리: 1.5주  
  - 분석: 1주  
  - Tableau 대시보드 제작: 1주  
- **기여도:** 100% (기획 → 데이터 수집 → 전처리 → 분석 → 시각화 전 과정 단독 수행)
- **결과 활용:** 완성된 대시보드를 기반으로 **마케팅 전략 기획서 작성 및 협업 실행 예정**

---

## 데이터 수집
- **수집 대상:** ‘와우솥뚜껑삼겹살(주 가게)’ + 인근 동일 업종 **경쟁 가게 7곳**
- **출처:** 네이버 지도 (Naver Place)
- **도구:** Python (Selenium, BeautifulSoup)
- **수집 항목:**
  - 업체명 / 주소 / 메뉴명  
  - 검색순위  
  - 리뷰 수 (방문자 리뷰, 블로그 리뷰)  
  - 간단 리뷰 수  
  - 텍스트 리뷰 (리뷰 본문, 방문일, 방문 횟수 등)

---

## 프로젝트 진행 과정
### 데이터 수집 및 정제
- Python을 활용해 네이버지도 리뷰를 크롤링  
- 중복 리뷰 및 불필요한 특수문자 제거  
- 날짜, 방문횟수 등 비정형 필드 정규화 처리  

### 텍스트 분석 (TF-IDF)
- 리뷰 텍스트를 형태소 단위로 분리  
- 불용어(stopwords) 제거 후 TF-IDF로 **가게별 핵심 키워드 도출**  
- 상위 등장 단어를 통해 각 가게의 특징적인 표현 비교  

### 감성 분석 (Sentiment Analysis)
- 자체 구축한 한국어 감정사전 + KNU 한국어 감성사전을 결합  
- 긍정어/부정어/중립어 분류 및 **가중치 부여**  
- 리뷰별 감정 점수를 계산 후, 가게별 평균 감성 점수 비교  

### 시각화 (Tableau)
- TF-IDF 및 감성분석 결과를 Tableau로 시각화  
- 지도 기반 위치 표시, KPI 지표, 키워드 트리맵 등  
- 사용자 인터랙션(마우스 오버, 클릭 등)을 통한 **동적 비교 기능 구현**

## 리뷰 데이터를 활용한 이유
- 매출, 방문 수 등 **상업적 민감 데이터 접근 제한**  
- 리뷰 데이터는 **공개 접근 가능 + 실고객 의견 기반**  
- 소비자의 실제 감정과 언어를 반영하므로 **경쟁력 분석 지표로 적합**
